Create an invite to sign for a merged document group

POST 

     
https://api-eval.signnow.com/documentgroup/{{group_id}}/groupinvite

Creates an invite to sign for a merged document group. group_id TBD. (ask developers about group id data type and how to find it)

Requirements (if any):

TBD. It is necessary to specify special requirements from the developers of the API.

Request

Headers

Authorization string (Required)

Bearer followed by token

Default: Bearer {{token}}

Body

Request example

Content-Type: application/json

curl --location --request 
POST 
'https://api-eval.signnow.com/documentgroup/{{group_id}}/groupinvite' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer {{token}}' \
--data-raw
'{
    "invite_steps": [
        {
            "order": 1,
            "invite_emails": [
                {
                    "email": "{{signer_email}}",
                    "subject": "Signer 1",
                    "expiration_days": 30

            "invite_actions": [
                {
                    "email": "{{signer_email}}",
                    "role_name": "Signer 1",
